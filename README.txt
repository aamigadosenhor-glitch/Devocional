Devocional 365 (site estático)

O que é
Um devocional diário que troca automaticamente, sem conta e sem envio de dados.

Botões
Compartilhar: abre o compartilhamento do aparelho, ou copia o texto.
Atualizar: limpa cache e recarrega, útil quando você altera textos e quer forçar a versão nova.

Música do dia no Spotify, dois modos
Modo playlist
1) Abra data/musicas.json
2) enabled true
3) mode playlist
4) cole o link da playlist em playlist

Modo tracks
1) enabled true
2) mode tracks
3) preencha tracks com links de faixas
Se tracks tiver menos que 365, o site repete usando o resto.

Formatos aceitos
Faixa: https://open.spotify.com/track/ID ou spotify:track:ID
Playlist: https://open.spotify.com/playlist/ID ou spotify:playlist:ID

Observação
O player do Spotify pode exigir login para tocar a música inteira, isso depende das regras do Spotify e do dispositivo.

Spotify e prévia
O player embed do Spotify pode tocar apenas uma prévia no navegador. Para ouvir completa, use o botão Ouvir no Spotify, ele abre a faixa ou playlist no app ou no site do Spotify.
